Chrome Multi Profile Launcher

크롬 다계정(.lnk) 프로필을 여러 개 동시에 실행하고
모니터 화면에 자동으로 깔끔하게 배치해주는 편의 프로그램입니다.

주요 기능

✔ 선택한 크롬 프로필 여러 개 동시에 실행

✔ 모니터 해상도에 맞춰 자동 레이아웃 배치
(Dead space 최소화, 정사각형에 가까운 배치 방식)

✔ 프로필 이름 자동 스캔
(*.lnk 파일을 숫자 기준으로 자연 정렬)

✔ 실행한 크롬 창만 부분 종료 기능 
(이건 아직 안 됨)

✔ 실행 옵션/배치 설정 자동 저장

✔ PyQt5 기반의 깔끔한 UI

❗주의사항

이 프로그램은 크롬 프로필을 실행하는 “바로가기(.lnk)” 방식으로 동작합니다.
Chrome이 PC에 설치되어 있어야 정상 작동합니다.

.lnk 안에 포함된 실행 경로가 틀리면 크롬이 열리지 않습니다.

🔧 크롬 프로필(.lnk) 만들기

프로필을 처음 만드는 경우 아래 방법으로 생성하세요.

크롬 오른쪽 상단 프로필 아이콘 클릭

프로필 → “⋮” → 바로가기 만들기

바탕화면에 생성된 .lnk 파일을
profiles/ 폴더로 옮기기

💡 파일명은 자동 인식되니 아무 이름이어도 됩니다.
